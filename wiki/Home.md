# KoKoFish — Audiobook Studio

<p align="center">
  <img src="https://github.com/kl3mta3/KoKoFish/blob/main/Images/icon.png" width="200"/>
</p>

KoKoFish is a local, offline-capable Audiobook Studio for Windows. Convert text, ebooks, and documents into spoken audio using multiple AI voices, clone your own voice, transcribe audio back to text, translate content into other languages, and use AI writing tools to polish your text before it ever hits the speaker.

Everything runs on your machine. No cloud. No API keys. No subscriptions.

## What Can KoKoFish Do?

- **Read any document aloud** — .txt, .pdf, .docx, .epub
- **54+ preset voices** across 10 languages (Kokoro engine)
- **Voice cloning** — record or upload a clip and create a reusable voice profile (Fish-Speech)
- **Blend two voices** together at any ratio (Kokoro)
- **Transcribe audio** to text using Whisper (7 model sizes)
- **Translate** text or audio into 12+ languages — locally, with no API
- **Re-read translated audio** — transcribe foreign audio, translate it, speak it in your target language
- **AI text tools** — grammar check, tone rewriting, TTS enhancement, tag generation, translation
- **Convert audiobook files** — combine chapters, export M4B, MP3, WAV, FLAC
- **Chat with a local AI** — Prompt Lab with multiple personality presets
